HTML(Hypertext Markup Language)

HTML was created by Tim Berners-Lee in late 1991 and released officially in 1995 as HTML 2.0 with the purpose of sharing documents. HTML 4.01 was published in late 1999 and was a major version. HTML has evolved with various updates and the recent version is HTML 5. HTML is used by web developers to instruct a web browser how content on the web will be placed and displayed as web pages.
